## 📚 What This Specialization Covers

- **Linear Algebra** — Vectors, matrices, eigenvalues, eigenvectors, and the Gram-Schmidt process
- **Multivariate Calculus** — Partial derivatives, gradient descent, optimization, and backpropagation math
- **PCA (Principal Component Analysis)** — Dimensionality reduction, covariance matrices, and projections
